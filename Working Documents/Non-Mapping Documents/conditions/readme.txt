This folder:
MARC2RDA\Working Documents\Non-Mapping Documents\conditions

...contains tools to analyze the conditions entered in the mapping documents.

The working report is conditionSheet.xlsx

It is created as follows:

1. run extractConditions.py
2. open conditionSheet.csv in Excel
    2a. sort as appropriate
          2a1. especially helpful to eliminate empty rows
    2b. Filter for unique records only
    2c. save as conditionSheet.xlsx
